# 인터페이스

## 인터페이스의 변수와 메소드
- 변수 => 상수형 변수 => public static final이 생략되어 있음
```java
int a=10;
```
- 메소드 : 추상화 메소드 => 구현이 안된 메소드이기 때문에 메모리 할당이 불가능하다.
```java
void display();
```

## 인터페이스의 특징
- 다중 상속이 가능하다
- 서로 다른 클래스 연결이 가능하다.
- extends : 인터페이스에서 인터페이스 상속
- implements : 인터페이스에서 클래스를 상속
