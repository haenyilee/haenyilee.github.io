# 사용자 입력창 : ```<input type="유형">```
#### 1. 입력(한줄)
- text (ID,이름..)
- password (비밀번호 입력창)
- file : 파일 업로드할때 브라우징하는 버튼
- email
- date
- hidden : 감춰두고 실행 (사용자는 못보는데 소스 상에는 존재하는 버튼)

#### 2. 버튼
- button : 일반 버튼
- submit : 전송 버튼 (데이터)
- reset : 초기화
- radio
- checkbox

#### 3. 여러개 입력
- textarea

#### 4. 목록
- select : 콤보박스
