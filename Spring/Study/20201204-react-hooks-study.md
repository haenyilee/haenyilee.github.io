# spring

### ReactController
- 리액트는 View의 역할밖에 수행하지 못한다.
- 리액트 : 데이터 관리 프로그램이라고 한다(state)
- 서버 프로그램 : Spring , NodeJS
- 두 서버 프로그램이 충돌되면 안된다.
- NodeJS는 Chat이나 실시간 데이터 전송을 할 때 주로 사용한다.
- Spring은 데이터를 보낼 때 사용한다.
- 사용자가 볼 수 있는 것 : Front
- 사용자가 볼 수 없는 것 : Back
